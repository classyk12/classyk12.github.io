Speaking detail

Smarter Features, Slower Apps: Lessons from Integrating AI in the Frontend

DevelopersUpNorth · Hull, UK · June 2026

Event overview

Shipping AI-powered UX without sacrificing speed

This session focused on the practical trade-offs teams face when adding AI features to production frontend systems. The talk covered performance bottlenecks, inference-aware UX design, and safe rollout patterns that protect reliability while improving user value.

  • How to identify and isolate AI-induced latency in user journeys.
  • Patterns for progressive enhancement and graceful fallbacks.
  • Monitoring strategies for AI-assisted frontend features in production.
